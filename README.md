# Ambianic Client UI

[![Greenkeeper badge](https://badges.greenkeeper.io/ambianic/ambianic-pwa.svg)](https://greenkeeper.io/)

Ambianic Client UI is an Offline-first Progressive Web App (PWA) to manage
Ambianic Edge deployments.
